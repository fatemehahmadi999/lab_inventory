# 🧪 Lab Inventory

A shared, real-time inventory and experiment tracker for the Winkler lab.

🔗 **[Open Lab Inventory](https://winkler-lab.github.io/lab_inventory/)**

> Password-protected. Contact the lab manager for the access code.

---

## What's in it

**Projects · Experiments · Materials · Tissues · Cell Lines · PDX Models**

All data is shared in real time across all devices. Changes save automatically. Works on mobile.

---

## Getting started

1. Open the link and enter the access code
2. Register your name the first time, then pick it from the dropdown on future visits
3. To remove a user: [Firebase Console](https://console.firebase.google.com) → Firestore → `users` → delete the document

---

## Tech

Single HTML file · Firebase Firestore + Storage · EmailJS · GitHub Pages
